주요 기능

🏢 멀티에이전트 팀

CEO, CMO, 콘텐츠 마케터, 퍼포먼스 마케터, 영업 담당 등 AI 직원을 자유롭게 구성합니다.

📱 텔레그램 원격 제어

이동 중에도 스마트폰으로 지시, 확인, 승인. 어디서든 회사를 운영합니다.

📋 이슈 기반 업무 관리

이슈를 만들고 담당자를 지정하면 AI가 자동으로 작업을 시작합니다.

🔔 실시간 알림

에이전트가 작업을 완료하거나 승인이 필요하면 텔레그램으로 즉시 알립니다.

텔레그램 가이드

명령어로 AI 팀을 제어합니다.

명령어설명
/iss이슈(업무) 목록 보기
/iss SOC-XX특정 이슈 상세 보기
/newiss새 이슈(업무) 만들기
/inbox알림 확인 — 에이전트 보고
/heartbeat에이전트 활동 시작
일반 메시지AI 비서에게 자유롭게 질문하거나 지시

5분 데모

이렇게 사용합니다.

1

텔레그램 연결

봇을 추가하고 /start 입력하면 끝.

나 /start
봇 연결 완료! AI 팀이 대기 중입니다.
2

현황 확인

한 마디로 회사 전체를 파악합니다.

나 지금 뭐하고 있어?
봇 CEO 대기 중, CMO 마케팅 전략 수립 중...
3

업무 지시

이슈를 만들면 AI가 알아서 일합니다.

나 /newiss → [CMO] → [마케팅]
나 인스타그램 마케팅 전략 기획해줘
봇 이슈 생성 완료!
4

결과 확인 + 승인

보고가 오면 확인하고 승인합니다.

봇 승인 요청이 들어왔어요~ [승인]
나 [승인] 클릭
봇 승인 완료!

설치

개발 환경 구축, 서버 세팅, 포트 설정 같은 건 없습니다.
터미널에 한 줄 붙여넣으면 AI 에이전트 팀이 구성됩니다.

Terminal
curl -sL https://geekalexy.github.io/install.sh | bash
필요한 것

Mac 컴퓨터 (항시 가동) + 텔레그램 앱 + Claude 구독 (Pro, Max 이상)

설치 과정

프로젝트명과 텔레그램 봇 토큰만 입력하면 됩니다. 나머지는 스크립트가 알아서 합니다.

자주 묻는 질문

프로그래밍 지식이 필요한가요?

전혀 필요 없습니다. 텔레그램으로 대화하듯이 사용합니다.

AI 직원은 몇 명까지 가능한가요?

CEO, CMO, 마케터, 영업 등 업종에 맞게 자유롭게 구성합니다.

비용은 얼마인가요?

오픈소스이므로 무료입니다. Claude 구독(Pro, Max 이상)만 있으면 됩니다.

모바일에서도 되나요?

텔레그램은 물론, 대시보드도 모바일에서 접속 가능합니다.
